AdvSidlister - Subdomain Finder Tool

Welcome to AdvSidlister, a powerful subdomain finder tool. Follow the instructions below to use the tool:

Step 1: Setup

Before using AdvSidlister, make sure you have set up the tool by running the following commands:

        chmod +x AdvSidlister.py
        ./AdvSidlister.py

Step 2: Running the Tool

To find subdomains for a domain, use the following command:

        ./AdvSidlister.py

Follow the on-screen prompts to enter the domain name and output file name.

Step 3: Viewing Results

After running the tool, you will have two output files:

  • Your subdomain results in an HTML file (e.g., subdomain_results.html).
  • A backup of the results in a text file (backup_output.txt).

Step 4: Analyzing Results

You can open the HTML file in a web browser to view the subdomain results in a table format.

Step 5: Exiting the Tool

To exit the tool, you can press Enter in the terminal.

Happy subdomain hunting with AdvSidlister!
